# Avani Health 개인정보 처리방침

최종 수정일: 2026-02-07

Avani Health(이하 "서비스")는 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 개인정보 처리방침은 서비스 이용 과정에서 개인정보가 어떻게 처리되는지 안내합니다.

## 1. 개인정보 수집 여부

서비스는 회원가입·로그인 기능을 제공하지 않으며, 이용자의 개인정보를 수집하거나 서버에 저장하지 않습니다. 이름, 이메일, 전화번호 등 개인을 식별할 수 있는 정보를 수집하지 않습니다.

## 2. 건강 데이터 처리 방식

이용자가 앱에 입력하거나 앱이 기기에서 수집하는 건강 정보(체중, 혈압, 걸음수 등)는 다음과 같이 처리됩니다. 모든 데이터는 이용자의 기기 내 로컬 저장소에만 저장되며, 외부 서버로 전송되지 않습니다. 서비스 운영자는 이용자의 건강 데이터를 열람하거나 수집할 수 없습니다.

## 3. 제3자 제공 및 외부 공유

서비스는 이용자의 데이터를 제3자에게 제공하거나 공유하지 않습니다. 광고 플랫폼을 연동하지 않으며, 분석 플랫폼(GA 등)을 사용하지 않으며, 외부 클라우드에 저장하지 않습니다.

## 4. 데이터 보관 및 삭제

이용자가 입력·수집한 데이터는 기기 내에만 저장됩니다. 이용자가 앱을 삭제하거나 앱 내 데이터 초기화를 실행하는 경우 해당 데이터는 삭제됩니다. 서비스 운영자는 이용자의 데이터를 별도로 보관하지 않습니다.

## 5. 개인정보 보호를 위한 조치

서비스는 이용자 데이터 보호를 위해 다음을 적용합니다. 데이터는 기기 내부 저장소에만 저장되며, 외부 네트워크로 전송하는 기능은 구현되어 있지 않으며, 운영자가 접근할 수 없는 구조로 유지됩니다.

## 6. 이용자의 권리

이용자는 언제든지 앱을 삭제하거나 데이터 초기화를 통해 저장된 정보를 삭제할 수 있습니다.
